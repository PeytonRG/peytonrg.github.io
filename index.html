  • SGA Elections
    November 2019 - January 2020
    Skills Used: C# SQL HTML CSS JavaScript Bootstrap

    Led a team of six co-op students in adding new features to Auburn University's SGA Election website to accommodate ranked choice voting, which was voted into effect by the student body in the 2019 SGA election. Much of the work in this update encompasses database schema changes to accommodate the new ranked choice voting method in addition to the existing "standard" method, which remains in effect for non-executive positions. A SQL engine also had to be created to calculate how ranked choice votes are allocated when candidates lose successive "rounds" during vote calculation. Finally, the page where students vote for candidates and the results and demographic reports for elections needed to be updated to reflect ranked choice voting results. My personal contribution to the project consisted of a complete rewrite of the front-end and back-end for casting votes to support both old and new voting methods. My redesign targeted a "mobile-first" user experience, so I created a user interface following paradigms of hierarchical menus for selecting candidates within an election position. Principal development on the update is nearing completion, and will be followed by final rounds of quality assurance, accessibility, and user acceptance testing.
